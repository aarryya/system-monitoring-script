# System Monitoring and Alerting Script

This project is a Bash-based system monitoring script that checks key system
resources and alerts when predefined thresholds are exceeded. It simulates
basic DevOps monitoring and automation tasks.

## Features
- Disk usage monitoring with alerts
- Memory usage monitoring with alerts
- Displays top CPU-consuming processes
- Displays top memory-consuming processes
- Logs alerts to a log file

## Usage
```bash
chmod +x system_monitor.sh
./system_monitor.sh
